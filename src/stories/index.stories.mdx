# Button

汎用BlockButtonコンポーネント。  
このコンポーネントの`Props` 確認できる

## Rounded

<Props of={BlockButton} />

Shape="rounded"なボタンですよ。  

<Preview>
  <Story name="Normal">
    {{
        components: { BlockButton },
        template: '<block-button shape="rounded" @click="action">Hello Button</block-button>',
        methods: { action: action("clicked") }
    }}
  </Story>
</Preview>
